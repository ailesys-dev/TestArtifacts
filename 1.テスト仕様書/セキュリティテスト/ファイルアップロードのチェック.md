### 禁止ファイルアップロードチェック

**前提条件**

- アップロード機能が利用可能である
- 対象システムがファイルアップロードに対応している

**手順**

1. 画面または API からファイルアップロード操作を行う
2. exe, js, php などの実行可能ファイルを選択しアップロードを試みる

**期待結果**

- アップロードが拒否される
- エラーメッセージが表示される、またはサーバ側でブロックされる

### アップロード可能ファイル拡張子の制御

**前提条件**

- アップロード画面または API の仕様が存在する

**手順**

1. 許可されていない拡張子のファイルをアップロードする（例：exe, bat, bin）
2. 許可されている拡張子のファイルをアップロードする（例：png, jpg, pdf 等）

**期待結果**

- 許可されていない拡張子はアップロードできない
- 許可された拡張子のみ正常にアップロードできる

### MIME タイプ検証

**前提条件**

- ファイル拡張子と異なる MIME タイプを偽装できるテストデータが用意されている

**手順**

1. png 拡張子だが中身が実行ファイルのファイルなど、MIME タイプ偽装ファイルを用意する
2. アップロード操作を行う

**期待結果**

- 拡張子ではなく正しい MIME タイプで検証が行われ、偽装ファイルは拒否される
